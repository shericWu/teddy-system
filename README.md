## ICG Final Project 8. Sketch system for animation (Teddy system)
### How to run?
[Install three.js - Option 1](https://threejs.org/manual/#en/installation)

TLDR:
- Install `node.js`
- Install `three.js` and `vite`
    - `$ npm install --save three`
    - `$ npm install --save-dev vite`
- Run
    - `$ npx vite`
